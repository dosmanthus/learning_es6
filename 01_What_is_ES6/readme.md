# What is ECMAScript 6?

- 1995 Javascript created.
- 1997 ECMAScript 1
- 2009 ECMAScript 5
- 2015 ECMAScript 6

# New key words

- let
- const
- arrow function

- tools: Babel.js/Traceur/Closure
